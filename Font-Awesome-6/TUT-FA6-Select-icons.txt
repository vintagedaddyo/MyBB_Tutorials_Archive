[Tutorial] - Font Awesome 6 select icons  - by Vintagedaddyo


1. Loading Font Awesome 6

In your Templates section of the MyBB ACP, find the Ungrouped Templates -> headerinclude template. Add the following after:

  {$stylesheets}

External linking:

  <link rel="stylesheet" href="https://use.fontawesome.com/releases/v6.0.0-beta3/css/all.css">

Internal linking:

  <link rel="stylesheet" href="{$mybb->asset_url}/font-awesome-6/css/all.css">

* You can download font-awesome-6 pkg stripped down for size to be used for theme includes here:

https://filetransfer.io/data-package/5mSLMORO#link


* Or you can also just download the latest Font-awesome-6 web files pkg from the Font-Awesome site and strip it down for size to be used for for internal theme include:

ie: download web-specific files source: 

https://use.fontawesome.com/releases/v6.0.0-beta3/fontawesome-free-6.0.0-beta3-web.zip

NOTE: You can skip the header include if you already have the font awesome 6 included in your headerinclude say from other parts of these tutorials.



2. CSS styling edits:


In your theme global.css:


Find:

select {
	background: #ffffff;
	padding: 3px;
	border: 1px solid #ccc;
	outline: 0;
	font-family: Tahoma, Verdana, Arial, Sans-Serif;
	font-size: 13px;
}


Replace with:


/** FA 6 Select icons by: Anka06 & Vintagedaddyo **/

select {
    padding: 4px;
    margin: 0;
    background: #FFF;
    border: 1px solid #ddd;
    color: #333;
    outline: none;
    font-size: 13px;
    display: inline-block;
    -webkit-appearance: none;
    -moz-appearance: none;
    appearance: none;
    cursor: pointer;
    width: 200px;
    border-radius: 0px;
    -webkit-border-radius: 0px;
    -moz-border-radius: 0px;
    
}

.select-wrap {
    position: relative;
}

.select-wrap:before {
    font-family: "Font Awesome 6 Free";
    font-weight: 900;
    content: '\f107';
    font-size: 15px;
    position: absolute;
    top: -2px;
    right: 10px;
    color: #c6c4c4;
    pointer-events: none;
}
